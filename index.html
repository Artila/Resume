关于我

我是居住在广州的前端开发工程师，广东外语外贸大学2016年计算机科学与技术的本科毕业生，到2017年为止，拥有一年左右的前端工作经验。

有着坚实的前端开发基础，能够熟练使用 HTML5 、CSS3 以及 JavaScript来构建网页；对于流行的框架，对于 jQuery 也不陌生，能够熟练做出各种特效，但在平时的日子，还是比较倾向于应用原生JavaScript；对于 Bootstap 也很熟悉，从最初构建小型网页到现在，从 Bootstrap 上也学到很多东西；能够保持对新技术的热情，最近在学习 Vue.js，这是一种非常奇妙，用起来非常便捷，高性能的框架，并且在实际开发中有所使用，例如我用Vue.js做了饿了么的购买页，具体项目可以在我的 Github 查看。在这个过程中，还学习了Node.js，Webpack，stylus 的用法。看了学习成果，觉得自己还很稚嫩，但确实一天天在成长。

对学习还保持着很大的热情。说起学习这件事，初学的时候，逛的比较多的是慕课网，接着是FCC——freecodecamp，再接下来的是Youtube，Github等等。其中也看到了各种程序员写的各种各样的文章，无论是技术还是生活。

爱代码，爱生活，目前的目标是做一名优秀的前端程序员。

经验

  • 2017

    自学

    在家

    从上一家公司之后，暂时没有找工作，反而沉迷于学习了。这段时间，最常泡的三个网站是慕课网——一个很多免费教学视频的网站，freeCodeCamp——一个自学网站，Github——全球最大的同性交流网站（笑）。目前最常用的是Github啦，每天点亮那个格子就是学习的动力啊，笑。对于 Github，我没有很多高大上的说法，他更多的作用是保管我的学习成果，便于复习，督促我学习。这样说一定很奇怪，但是当你每天坚持上传代码，这样的结果就是你每天都会学习一点新东西，一旦停下来你就会觉得浑身不自在。而且也不怕代码找不到了。

  • 2016

    前端开发

    菠萝合租

    这段时间是真正独挡一面的阶段，因为这个时候，很多工作都是自己一个人完成，大概除了交流的视觉设计师和产品经理，前端开发的前辈任务也很重。在这个阶段，就算费劲心思，用尽所学，有时候也满足不了需求，所以在下班的时候，很大程度还要为自己充电，在网上翻看各种教程，视频，学习心得。在这个过程中，接触到很多新技术，但是我发现往往这一切的解法很多都回归于最原始，最基础的技术。
